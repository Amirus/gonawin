Gonawin themes
===================

naming convention:
     
     [adjective-plant]

please detail:

 * the colors you customize
 * the fonts you use
 * images of main page, teams page, team page and user page.
 
how to add a new theme:

 * copy main_crazy_rose.css
 * edit your CSS with your values
 * set your css file in index.html
